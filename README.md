# Kiuli

Luxury African Safari travel website.

## Quick Start

```bash
npm install
npm run dev
```

- **Frontend:** http://localhost:3000
- **Admin:** http://localhost:3000/admin

## Production

- **Website:** https://kiuli.com
- **Admin:** https://admin.kiuli.com

## Documentation

- `CLAUDE.md` — Project instructions for AI assistants
- `SYSTEM_ARCHITECTURE.md` — Technical architecture overview
- `DEPLOYMENT_CHECKLIST.md` — Deployment procedures
- `KIULI_LAMBDA_ARCHITECTURE.md` — Scraper pipeline details

## Stack

- Next.js 15
- Payload CMS 3.63
- Tailwind CSS 3.4
- Vercel (hosting)
- AWS Lambda (scraper pipeline)
- AWS S3 + imgix (images)
